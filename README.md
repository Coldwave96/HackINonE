## 安装使用
⚠️ ** 注意：必须以root权限运行 HackINonE ！！！
```bash
git clone https://github.com/Coldwave96/HackINonE.git
sudo chmod -R 755 HackINonE
cd HackINonE
sudo pip3 install -r requirement.txt
./install.sh
sudo HackINonE
```
